# Security Policy

Please do not include private dictionary files, Google Japanese Input exports, API keys, local paths, screenshots, account data, customer names, or personal names in public issues.

If you believe sensitive data was exposed, contact the maintainer privately instead of opening a public issue with the data.

This project does not require API keys. If an agent runtime or Computer Use environment asks for credentials, treat that as separate from this repository and do not commit those credentials here.
