# Privacy And Safety

This project is designed to keep private dictionary data out of public repositories.

Do not commit:

- real dictionary files,
- Google Japanese Input export TSVs,
- backup directories,
- run logs,
- screenshots,
- local paths,
- account data,
- API keys or tokens,
- customer names or personal names.

`出典` is metadata only. If you plan to share a source file, do not put private URLs, local paths, account names, ticket IDs, customer names, or confidential source names in that column.

Computer Use is optional. If you use it, keep it scoped to Google Japanese Input and require explicit confirmation immediately before changing a real dictionary.

The project is not affiliated with, endorsed by, or sponsored by Google.
